ЗАДАНИЕ #9. Spring Security:
Необходимо доработать приложение из задания №8:

+1) добавить механизм аутентификации пользователей на сайте с использованием логина и пароля;

+2) добавить механизм аутентификации при работе с API с использованием токена (jwt);

+3) добавить три группы ADMIN (полный доступ ко всем функциям), OPERATOR (доступ к отчетам и получению данных по
   количеству счетчиков в группах) и METER (только передача данных);

+4) ограничить доступ к функционалу на основе правил п. 3;

+5) для инициализации базы данных первой учетной записью администратора использовать механизм миграций (Liquibase);

+6) добавить раздел на сайт для работы с пользователями (+просмотр, +добавление, +изменение, +включение в группу, +блокировка);

7) реализовать функционал "запомнить пользователя на сайте", чтобы не вводить пароль каждый раз.
